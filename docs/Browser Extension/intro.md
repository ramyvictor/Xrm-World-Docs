# Introduction
Xrm-World Extension is a browser tool available for **Chrome** and **Edge** that enhances productivity when working with **Microsoft Dynamics 365**.

Originally released in **2018**, this extension adds useful tools directly inside the browser when working with CRM instances — such as quick navigation, record IDs, and form tools.

The extension is **open source** and designed to work in both sandbox and production environments.

---

# Features
- Toggle advanced find
- Quickly access record IDs
- Form tab navigation tools
- Open classic settings
- Tools for developers and power users

For more details, visit the [original blog post](#).

---

# Getting Started

### 1. Install the Extension
Choose your browser:
- [Chrome Web Store](#)
- [Microsoft Edge Add-ons](#)

Click **Add to Chrome** or **Get** to install.

### 2. Pin the Extension
After installation:
- Click the puzzle icon in your browser toolbar.
- Pin **Xrm-World** to make it always visible.

### 3. Open Your CRM
Go to your **Dynamics 365** environment.
The extension auto-detects that you're in a Dynamics CRM app and activates.

### 4. Use the Toolbar
Click the extension icon to open the toolbar:
- Use built-in actions like copying the record ID.
- Switch forms or tabs easily.
- Access developer tools relevant to your current view.
